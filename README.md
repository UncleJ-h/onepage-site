- `index.html` — Product landing page (Chinese)
- `index-en.html` — Product landing page (English)
- `privacy.html` / `privacy-en.html` — Privacy Policy
- `support.html` / `support-en.html` — Support & FAQ

## Deploy

This site is designed to be hosted on GitHub Pages.

```bash
# Push to GitHub and enable Pages in repo settings
git add .
git commit -m "Initial website"
git push origin main
```
